# Diretrizes de Frontend Premium — ORQUESTRA

## Objetivo

Criar um frontend administrativo premium, moderno e sofisticado para o projeto **ORQUESTRA**.

A interface deve transmitir:

* autoridade
* organização
* tecnologia
* sofisticação
* controle
* clareza
* experiência high-end

O sistema deve parecer uma plataforma executiva moderna, refinada e estratégica.

---

# Conceito Visual

A estética do ORQUESTRA deve seguir uma linha:

* dark premium
* minimalista
* tecnológica
* elegante
* futurista
* executiva

A experiência visual deve lembrar:

* fintechs premium
* plataformas SaaS modernas
* dashboards high-end
* sistemas estratégicos corporativos

---

# Direção de Identidade Visual

## Tema Principal

### Dark Mode Premium

O sistema deve utilizar predominantemente:

* preto sofisticado
* grafite profundo
* tons escuros refinados
* superfícies translúcidas
* contrastes suaves

Evitar:

* preto absoluto excessivo
* interfaces agressivas
* visual gamer
* aparência neon exagerada

# Componentes Visuais

## Cards Premium

Características:

* cantos suaves
* sombras discretas
* fundo sofisticado
* glassmorphism leve
* hover elegante
* profundidade visual

---

## Tabelas

As tabelas devem:

* possuir hierarquia visual clara
* usar espaçamento confortável
* evitar excesso de linhas
* possuir filtros refinados
* utilizar badges modernas

Evitar:

* aparência de ERP antigo
* tabelas densas
* excesso de bordas

---

## Formulários

Os formulários devem:

* possuir bastante respiro
* organização em etapas
* validações suaves
* UX limpa
* foco em clareza

Adicionar:

* estados vazios elegantes
* feedback visual refinado
* skeleton loading
* transições suaves

---

# Glassmorphism

* navbar
* modais
* dropdowns
* widgets especiais
* cards estratégicos

Evitar exageros.
